# Contributing

Thanks for your interest in contributing! This project follows the standard
fork-and-pull-request model.

## Development setup

1. Install Rust (stable) and `just`: `cargo install just`
2. Clone the repo
3. Run `just ci` to verify all checks pass locally

## Pull request process

1. Fork the repo and create a branch from `main`
2. Make your changes; ensure `just ci` passes
3. Update the CHANGELOG.md if your change is user-visible
4. Open a PR; address review feedback
5. Squash-merge once approved

## Code of conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/).
