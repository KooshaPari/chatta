# chatta

Real-time chat application with SvelteKit frontend and backend services.

## Quick Start

```bash
cd frontend && npm install && npm run dev
cd backend && npm install && npm run dev
```

## Dependencies

- Node.js 20.x
- SvelteKit 2.x
- TypeScript 5.x
- VSCode JSON-RPC for backend communication
